Mira Translator | 沉浸式 YouTube 翻譯

沉浸式網頁全文翻譯与 YouTube™ 雙語字幕增強工具
ウェブサイト翻訳と YouTube™ 2ヶ国語字幕表示ツール
Open Source | Ultra-lightweight (0.3MB) | Privacy-First
開源項目 | 極致輕量 (0.3MB) | 隱私至上
オープンソース | 超軽量 (0.3MB) | プライバシー重視
Add to Chrome | 免費安裝 | 無料でインストール
我厭倦了那些臃腫、需要註冊、到處追蹤數據的翻譯插件，更厭倦了那些把「開始使用」變成一場設定噩夢的工具——於是自己動手做了 Mira。
這裡沒有伺服器，沒有會員牆。代碼完全開源，支持自定義 API 密鑰。打開即用，你的數據只屬於你。
I was tired of bloated, account-gated tools that treat your data as their product. So I built Mira: a 0.3MB, serverless, and open-source translator. No setup nightmares, no tracking. Your words, your keys, your privacy.
肥大化したツールや登録の強制にうんざりし、設定が「地獄」にならないツールを求めて Mira を開発しました。サーバーなし、広告なし。オープンソースで独自の API キーも利用可能。あなたのデータは、あなただけのものです。

🌟 Core Features (核心亮點 / 機能紹介)

Visual Element Picker (可視化拾取 / ビジュアル選択)

Precisely define translation areas with a single click. 直感的クリックで翻訳範囲を自由に設定。

Innovative Learning Flow (創新學習流 / 学習フロー)

Instant word lookup on YouTube™ subtitles, synced with Google Drive. 字幕単詞をダブルクリックで即座に検索、Google ドライブと同期。

Personal Vocabulary Book (個人生詞本 / 単語帳)

Every word you look up is saved with its definition, part of speech, phonetics, source context, and timestamp — stored privately in your Google Drive. Export to CSV anytime.
查過的每個單字自動收入生詞本，附音標、詞性、來源上下文，存於你的 Google Drive，隨時匯出 CSV。
調べた単語を音標・品詞・文脈とともに自動保存。Google ドライブに同期し、CSVエクスポートも可能。

High-Quality Cache (智能緩存 / AIキャッシュ)

Smartly locks high-quality AI translations to save API costs. 高品質AI翻訳をスマートにキャッシュし、APIコストを節約。

Privacy Centric (隱私至上 / プライバシー重視)

Serverless architecture. Private sync via your Google Drive. サーバーなしの設計。Google ドライブ経由でプライベートに同期。

📸 Gallery (功能圖賞 / ギャラリー)

1. Immersive Video Learning (沉浸式影片學習)

Elegant bilingual subtitles and instant lookup on YouTube™. YouTube™での優雅な2ヶ国語字幕と即時単詞検索。

2. Instant Style Editor (即時樣式編輯 / WYSIWYG)

True WYSIWYG: No need to go back and forth between settings pages. Modify colors, underline styles, and highlights with instant visual feedback on the same page. Apply independent configs for both Website Translation and YouTube™ Subtitles with just one click.

所見即所得： 徹底告別繁瑣的設置頁面往返。直接在當前頁面定義顏色、底線樣式与高亮范围，點擊即刻應用，效果實時呈現。支持 網頁翻譯 与 YouTube™ 字幕 獨立配置。

WYSIWYG 編集： 設定画面との往復はもう不要。テキスト色、下線スタイル、ハイライト範囲を定義し、クリックで即時反映。ページ上の翻訳結果をリアルタイムに確認。ウェブ翻訳 と YouTube™ 字幕 の独立設定に対応。

2. Precision Visual Picker (精準可視化拾取)

Define translation areas on any website with one click. クリック一つでウェブサイトの翻訳範囲を設定。

3. Personal Vocabulary Book (個人生詞本 / 単語帳)

Auto-saved vocabulary with phonetics, definitions, and source context. Export to CSV anytime.
自動保存生詞，附音標、釋義與來源語境，隨時匯出 CSV。単語を自動保存、いつでもCSVエクスポート。

4. Smart Engine & Private Sync (智能引擎與私密同步)

Powerful AI persistent cache and Google Drive sync. 強力なAIキャッシュとGoogle ドライブ同期。
